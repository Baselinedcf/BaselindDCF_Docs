# Glossary

This glossary defines common terms for integration purposes. It avoids formulas and engine implementation details.

| Term | Definition |
| --- | --- |
| Property | The real estate asset being modeled. |
| Project | The top-level configuration for a property model, including property type, analysis dates, area, currency, and general assumptions. |
| Analysis start | The month and year where the model timeline begins. |
| Holding period | The modeled investment duration. |
| Tenant | A commercial rent-roll record representing an occupied, vacant, or speculative leased space. |
| Unit type | A grouped unit category for unit rental properties, such as an apartment floor plan or self-storage unit size. |
| Market leasing category | A reusable set of leasing assumptions assigned to tenants or space. |
| Recovery method | A named reimbursement structure for commercial expense recoveries. |
| Income line | A non-base-rent income item. |
| Expense line | An operating expense item. |
| CapEx line | A capital expense item. |
| Debt | Loan or financing assumptions when supplied. |
| Resale/PV assumptions | Inputs related to present value and exit modeling. |
| Schema version | Version of the input or result data contract. |
| Engine version | Version marker for the calculation engine that produced results. |
| Embedded inputs | JSON inputs stored inside an exported workbook. |
| Embedded outputs | JSON report results stored inside an exported workbook. |
| Audit manifest | Metadata stored with embedded workbook JSON, including version and hash information. |
| NOI | A system-computed income metric used in reporting and valuation outputs. |
| CFBD | A system-computed cash flow metric before debt service. |
| CFAD | A system-computed cash flow metric after debt service. |
| Terminal value | A system-computed or input-driven exit value reported by Baseline. |
| Net reversion | A system-reported exit proceeds metric. |
